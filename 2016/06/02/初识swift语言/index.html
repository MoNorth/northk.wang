初识swift语言

发表于 2016-06-02   |   分类于 swift   |  

由于好奇，就看看了传说中的swift语言。花费了不少脑筋才看完了基础语法部分，实在汗颜

基础语法

  1. 变量/常量声明
    swift是一门强类型的语言，可是他的变量声明基本都是用var，要不是我学习过c#和javascript，还真有点接受不了。在c#和javascript中也可以使用var关键字来声明变量，不过作用完全不一样。在js中，用var来声明所有类型的变量，因为它是一个弱类型的语言，不同类型的变量之间也可以相互赋值。但在c#中，var是用来推断类型的，意思就是编译器根据赋的值来确定这个变量的类型，推断之后，类型就确定了，就不能再给他赋予别的类型的值了。swift的var的用法和c#中的一样，属于推断类型。
    在swift中还可以用let关键字来声明常量，这个和es6中的let也大不一样，它也是推断类型，不过是常量，不可更改。
    当然也可以在变量声明时确定变量类型，使用:类型。例如声明一个字符串类型var str:String。
  2. 字符串和字符
    在swift中，和大多数语言一样，字符串并不是字符数组。所以他不能通过下标的方式来访问字符串中的字符（js都行。。）。要取值的话当然也有其他方式，这里我介绍一种xcode6.2中的方法str[advance(str.startIndex, 2)]。
    还有一点很重要，字符串是值类型。
  3. 整形和浮点型
    在swift中，可以直接声明Int, Float, Double类型的变量，他们默认的字节数是以操作系统为准的。也有指定字节长度的声明方式Int8, Int16, Int64,Float32, Float64等，Float64其实就相当于Double，使用推断类型的话，整形默认是Int,浮点型默认是Double。
  4. 可选类型
    我觉得也就是c#中的可空类型。声明方式为var num:Int?。即可为空值,空值表示为nil。在调用其方法是可使用num?.func，其表示当num不为空的时候调用。也可以强制取值，将其当作一般类型num!,不过如果为空会出错。
  5. 元组
    我个人觉得swift的元组是一大利器，声明方式为var demo = (a:1, b:"hello"),其中可有键值a, b存在，也可以不存在。如果存在，取值方式为demo.a或者demo.0,如果不存在，只可通过demo.0取值。也可以这样声明元组：var (a, b) = (1, 2),这样取值可直接去a或b;

运算符

  1. 基本运算符
    和别的语言没什么区别，多了一个...和`..<的区间运算符。如1...10，表示1——10中的所有数，包括1和10。1..<10不包括10.

  2. 运算符重载
    比如两个特殊类型（Point）的加法运算

  3. 自定义运算符
    方式类似于运算符重载，不过必须先声明运算符，还可以自定义运算符特性

复杂数据类型

  1. 数组
    和其他面向对象语言一样，同一数组只能拥有同一种数据类型，声明方式var array = [Int](),var array = []。
  2. 字典
    和其他面向对象语言一样。声明方式var dic = Dictionary<String,String>()

  3. 结构体
    和c语言结构体大致相似。

条件控制语句

和其他语言一样，该有的都有。这里说两个

  1. for…in…
    使用方式基本类似于js中的，只不过for关键字后面跟的是值而不是键。还可以直接在后面跟一个元组for (key, value) in array。还可跟_,表示不取值，值循环。通过_访问不到任何值。
  2. switch
    强大的简直可怕，在case后可跟常量，变量，元组等，具体使用方式请自行研究。提一点。就是在swift中，switch遇到case捕获后，即可中断，不需要添加break,如果需要类似于c语言中的效果，可在case语句块中添加fallthrough。

函数

类似于js中的函数，可返回任何东西，数字，字符串，函数，元组等。有一点提一下，在他的形参中，可拥有参数的提示键。也可没有。如果提示键要使用变量名，可在变量名前加#。

在swift中有一点不同，就是如上所述声明的函数中，所有实参不可更改，如要更改，及要加上var声明。如果更改后要影响原值(即指针传递),形参前要加inout,传参时要加&。注意：在形参中var,let,inout只能同时用一个。

闭包

如果我没有学过c#和javascript，那么我的闭包绝对理解的没这么快。在swift中，闭包邮电类似于c#中的lamda表达式,即类似于匿名函数，可用于函数传参，函数返回值等。它也可引用父级元素中的变量，并即使父级元素释放，也仍能引用，这点类似于js中的闭包。
