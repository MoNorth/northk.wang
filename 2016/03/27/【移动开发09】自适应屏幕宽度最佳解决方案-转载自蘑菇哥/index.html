自适应屏幕宽度最佳解决方案

本文把双11实践过的自适应屏幕方案总结了一下。
特别感谢下@特尼学长，本文中很多技术点都是@特尼学长琢磨出来的。

A – 用width，margin百分比和border-box属性布局商品列表

1) PSD量好商品宽度，计算好宽度百分比，magin百分比来布局图强。

2) 如果商品容器有边框则要设置border-box属性，避免边框宽度影响图强布局照成误差。

CSS代码

/*重置盒子模式*/
.border-box {
  -webkit-box-sizing: border-box;
  -moz-box-sizing: border-box;
  box-sizing: border-box;
}

/*商品容器*/
.goods_box_list .goods_box {
    float: left;
    margin: 0 0 2% 2%;
    width: 47%;
    border: 1px solid #000;
    background: #fff;
    position: relative;
}

PHP代码

<?php
// 测试图强 20个商品
for( $i=0 ; $i < 20 ; $i++ ) {
?>
<div class="goods_box border-box fl">
    。。。。。。。。
</div>
<?php } ?>

B – 用padding-bottom百分比来做图片容器等比宽高变化

1) height直接设置为0，用padding-bottom百分比等于容器高度除以父级宽度。

2) 因为高度是百分比算的，由于小数四舍五入的缘故，容器高度可能会比图片实际高度小1px。
所以要加overflow:hidden防止容器被撑开，破坏布局。

CSS代码

/*填充图片样式*/
.fill_img { 
  display: block;
  width: 100%;
}

/*图片容器样式*/
.goods_box_list .goods_img_box {
    width: 100%;
    height: 0;
    padding-bottom: 150%;
    overflow: hidden;
}

PHP代码

<div class="goods_box border-box fl">
    <?php // 图片容器 ?>
    <div class="goods_img_box">
        <img class="fill_img" src="http://dummyimage.com/400x600/a3c0b9/fff">
    </div>
    。。。。。。。。。
</div>

C – 用rem来做字体大小、文本行高自适应

1) 先安照市面上的屏幕宽度用media来给html设置对应的基准字体。

2) body应该把font-size属性重置会12px，防止页面里默认字体大小乱掉。

3) 基准字体不能小于12px，因为部分浏览器不支持12px一下字体，举例10px的基准字体会被默认当12px来计算，
所以最小基准值取100px,而不是10px或1px。

参考资料 http://www.w3cplus.com/css3/define-font-size-with-css3-rem 

/*320默认rem字体标准值*/
html{font-size:100px;} 
/*body上把默认字体大小重置回来*/
body{font-size: 12px;}

/*media rem字体适配*/
/*针对360px以上的屏幕做适配*/
@media (min-width: 360px) {
  html{font-size:112px !important;}
}
/*针对400px以上的屏幕做适配*/
@media (min-width: 400px) {
  html{font-size:125px !important;}
}
/*适配480px以上屏幕*/
@media (min-width: 480px) {
  html{font-size:150px !important;}
}
/*适配540px以上屏幕*/
@media (min-width: 540px) {
  html{font-size:168px !important;}
}
/*适配640px以上屏幕*/
@media (min-width: 640px) {
  html{font-size:200px !important;}
}

4）在需要的文本元素样例使用rem单位。取值为相对应当前屏幕宽度的根元素(即html元素)font-size计算值的倍数。

举个列子，样例里的商品名称要求在[320px,360px)宽度里，字体大小为12px，行高和高度都为18px。

CSS代码

.goods_box_list .tgoods_name {
    font-size: 0.12rem;
    line-height: 0.18rem;
    height: 0.18rem;
    display: block;
    font-weight: bold;
    overflow: hidden;
    color: #222;
}

PHP代码

<div class="goods_box border-box fl">
    <?php // 图片容器 ?>
    。。。。。。。
    <?php // 商品信息容器 ?>
    <div class="tgoods_info">
        <?php // 商品名称 ?>
        <p class="tgoods_name">SIXGOD劲凉花露水，蓝胖专用香水！！！</p>
        <?php // 商品价格和折扣 ?>
        。。。。。。。
    </div>
</div>
